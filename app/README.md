# App Workspace

这个目录是项目的前后端运行工作区。

更完整的项目说明、功能概览、接口说明和运行方式，请查看仓库根目录 README：

- [../README.md](../README.md)

## 常用命令

```bash
npm install
npm run dev
npm run build
npm run lint
npm run start:server
```

## 说明

- `npm run dev` 会同时启动前端和后端
- 前端默认地址是 [http://127.0.0.1:5173](http://127.0.0.1:5173)
- 后端健康检查是 [http://127.0.0.1:8787/health](http://127.0.0.1:8787/health)
